```cpp
for (const auto &person : present_people) {
        cout << person << "\n";
}
for (const auto &person : present_people) {
        cout << person << endl;
}
```

- “\n”의 경우 endl보다 효율적이고 시간을 단축시킨다!!
- endl의 경우 스트림 버퍼의 끝에 \n을 삽입해주고 flush까지 수행하여 버퍼를 비워준다.
- 버퍼를 비운다는 건 버퍼에 저장돼있던 내용을 내보낸다는 거다. stdout 버퍼를 비우면, 화면으로 그 내용이 출력된다. 이를 flush라고 한다.
